## Getting Started

First, add the project dependencies:

```bash
npm install
# or
yarn
# or
pnpm install
# or
bun install
```

Second, run the fake API server:

```bash
npm run server
# or
yarn server
# or
pnpm server
# or
bun server
```

So, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```



Finally, open [http://localhost:3000](http://localhost:3000) with your browser and enjoy it!
